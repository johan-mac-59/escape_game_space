# 🎮 Escape Game en Python

Un jeu d'escape game textuel structuré par salles, développé initialement en équipe et optimisé de manière autonome.

## 📝 Description du projet
Le but du jeu est de s'échapper de différentes salles successives en résolvant des énigmes et en trouvant des codes. Le joueur dispose d'un nombre de tentatives limité pour chaque énigme, ajoutant une mécanique de stress et de logique.

Ce projet a été réalisé en collaboration avec 2 collègues de formation, puis **entièrement revu, uniformisé et optimisé par mes soins** (refactoring du code, gestion des exceptions, harmonisation de la syntaxe).

## 🚀 Fonctionnalités clés
* **Système de progression par salles** avec des énigmes uniques.
* **Gestion des mécaniques de jeu :** Compteur de tentatives, conditions de défaite et de victoire.
* **Code uniformisé :** Structuré pour être facilement lisible et maintenable (fonctions typées, gestion propre des entrées utilisateur).

## 🛠️ Technologies utilisées
* **Python 3.x** (Logique métier, structures conditionnelles, boucles).
* **HTML5 / CSS3 / JavaScript :** Une version web interactive (`.html`) est disponible dans le dépôt. Elle a été générée via une IA (Claude.ai) sur la base de la logique de mon code Python, permettant de tester le jeu directement dans un navigateur.

1. Clonez le dépôt ou téléchargez le fichier `.py`.
2. Ouvrez votre terminal dans le dossier du projet.
3. Lancez la commande suivante :
   ```bash
